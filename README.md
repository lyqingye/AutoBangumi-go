该项目为个人使用的一个自动追番工具，其主要功能是从其它资源网站收集种子文件，通过重命名后推送给下载软件进行下载，文件通过重命名后可以直接被jellyfin直接识别  
通过配置要看的番剧名称，实现自动追番

该项目的宗旨是解放双手  

**注: 该项目永不收费, 用爱发电, 仅仅为了实现看番自由**

# 功能
- 支持Mikan站点
- 支持Qb下载器
- 支持订阅和补齐番剧
- 自动文件重命名
- 遵循分辨率与字幕语言优先规则
- 自动重新下载画质更高的剧集
- 支持使用TelegramBot 搜索番剧，然后自动追番
